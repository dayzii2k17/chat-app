Informações
Chat P2P que funciona em rede local (LAN) conectando 2 ou mais dispositivos utilizando de Sockets em java. Para isso um dos dispositivos precisa ser o Servidor fornecendo um nome, uma porta (algumas portas são do sistema...), por padrão a 9898, e o endereço IP do dispositivo que é adivinhado pelo timey wimey. Depois de criado o server basta criar o client fornecendo o IP e a porta do server já criado. Não é preciso conexão com a internet, no entanto a rede que os dispositivos compartilham precisa ser estável.
